# QuizzApp

## Beschreibung

Die App ist eine einfache Quiz-App: Man beantwortet einfach mehrere Fragen mit jeweils drei Antwortmöglichkeiten. Nach jeder Antwort zeigt die App an, ob die Antwort richtig oder falsch war.

## Funktionen

* Startbildschirm
* Mehrere Quizfragen
* Drei Antwortmöglichkeiten pro Frage
* Feedback nach jeder Antwort
* Punktestand während des Quiz
* Endbildschirm mit finaler Punktzahl
* Möglichkeit, das Quiz neu zu starten

## Beispiel einer FRage

Frage:

Welche Programmiersprache wird oft für Android-Apps genutzt?

Antwortmöglichkeiten:

* Kotlin
* Python
* HTML

Bei richtiger Antwort wird der Punktestand erhöht.

## Verwendete Technologien

* Kotlin
* Jetpack Compose
* Android Studio
* Material Design 3
